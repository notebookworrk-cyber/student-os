# Student OS

Your personal study productivity OS. Track focus, tasks, XP and level up.

## Tech Stack

- **Frontend:** Vanilla JS (ES Modules)
- **Build:** Vite 5
- **AI:** Google Gemini API integration
- **PWA:** Service worker + manifest
- **Theming:** VisionOS Liquid Glass design

## Setup

```bash
npm install
npm run dev
```

## Features

- **Focus Timer** — Pomodoro/Short/Long break modes with ring display
- **Task Management** — Subject tags, priorities, filter tabs
- **AI Chat** — Gemini-powered study assistant
- **Gamification** — XP, levels, achievements, weekly heatmap
- **Deep Focus** — Full-screen distraction-free mode
- **Statistics** — Weekly charts, habit scores, streaks

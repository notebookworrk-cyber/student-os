# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 2.x     | ✅        |
| 1.x     | ❌        |

## Reporting a Vulnerability

Report vulnerabilities via [GitHub Security Advisories](https://github.com/notebookworrk-cyber/student-os/security/advisories/new).

Do not open public issues for security vulnerabilities.
